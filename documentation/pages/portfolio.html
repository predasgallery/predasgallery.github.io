The portfolio

This page will guide you through the portfolio feature.

HTML Structure

For any portfolio grid you add, the HTML will look something like this:

<div class="grid">
  // Grid sizer. Removing this will break the layout!
  <div class="grid-sizer"></div>
  // The portfolio item
  <a class="grid-item">
    // The background
    <div class="thumb" style="background-image: url(path/to/image);">lt;/div>
    // The caption
    <div class="caption">
      <div class="type"> Type of work </div>
      <div class="description"> 
        <h4>Title</h4>
        <p>Description</p>
      </div>
    </div>
  </a>   
  // A second item
  <a class="item" ...
</div>

As you can see it's a background image that's showing the thumbnail image. This way, you can use any size and any proportion of the image in your portfolio. No need to resize! Plus, it's retina ready when you add a higher resolution image.

Item proportions

You can easily change an item's proportion by changing it's class. For example:

<div class="grid">
  // The portfolio item
  <a class="item wide">
    ...

As you can see it's now using a ".wide" class, which gives the item twice the width. You could also set the class to ".tall", which gives it twice the height!
