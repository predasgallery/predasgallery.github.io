Tera slider 3.1

This page will explain the Tera Slider

HTML Structure

The Tera slider is lightweight and has an easy-to-use HTML structure, as follows:

<div class="slider">
  <ul class="slides">
    <li class="slide">
      // Content slide 1
    </li>
    <li class="slide">
      // Content slide 2
    </li>
  </ul>
<div>

Knowing this, you can add as many slides as you want.

Slide background

The slide background is actually a div object that owns a style property with the corresponding background image url. Just like this:

// Inside .slide > section.hero
<div class="bg" style="background-image:url(url-to-image)"></div>

Dark mode for dark backgrounds

For light on dark slides, Be has a special class you can add. Add the ".dark" class to the <li class="slide"> element inside the slider to change the colors of the text and header to white when the active slide has this class.

Carousel mode (new in 3.1)

To activate carousel mode, add the .carousel class to the .slider and set the visible items by using the following attribute: "data-visible" = "4" (which can be any number).

Indicators (new in 3.1)

If you would like to show indicators of the slider or carousel, just add the "indicators" = "true" attribute to the .carousel element. See an example on the product.html page.
