Cheatsheet

Want to know what class does what? Here you can see a list of all helper (and handy) classes packed with Bone.

Helper classes

CTR+F (WIN) / CMD+F (MAC) this page if you are looking for a specific class.

.vbottom Aligns elemts to the bottom. Used on hero elements, see the index.html for and example.
.vcenter Center aligns the elements vertically.
.text-center Centers the text horizontally.
.offset-top Adds an offset to the top with 15% of the viewport height.
.gray Adds a light gray background
.white Defaults a background to white.
.top-line This shows a border at the top of a section.
.bottom-line This shows a border at the bottom of a section.
.dark-hero When used on the body tag, this class will set the content (of a hero) to white, including the logo. Also automagically added by the Tera Slider when needed.
.dark Gives a dark background and white text. Can also be used on Tera Slider slides, so the script knows when to change the logo to a white version.
.bold Changes the font weight of the content inside this div or section to bold.
.title-block Adds spacing for title blocks. Can be quite handy!
.boxed Use this on the body if you need a boxed style website!

Element classes

.hollow Use this on a button to make it look hollow (border only)
.slider This activates the Tera Slider, use .slides and .slide as children.
.bg Use this class and inline background:url(image/path) to have a background image.
.faded Use this on a .bg element in which image needs to be faded
.faded-subtle When the .faded class adds too much fade, you can use this subtle class.
.carousel Adds carousel functionality to the Tera Slider

Icon classes

.fa Use this on every FontAwesome icon class. FontAwese Cheatsheet
.ion-* Use this on every Ion icon class. Read more about Ionicons
